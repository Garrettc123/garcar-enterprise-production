API Documentation

Full interactive docs are available at /docs on the backend. Below is a quick reference.

Authentication

All product endpoints require either a Bearer token (from login) or an API key header.

# Bearer token
curl -H "Authorization: Bearer YOUR_TOKEN" ...

# API key
curl -H "X-API-Key: YOUR_API_KEY" ...

Base URL

http://localhost:8000

Registration

POST /api/auth/register
{
  "email": "you@company.com",
  "password": "your-password",
  "name": "Your Name"
}

Returns: { "access_token": "...", "user": { "api_key": "gce_..." } }

Deal Desk — Analyze

POST /api/products/deal-desk/analyze
{
  "company_name": "Acme Corp",
  "deal_size": 50000,
  "stage": "proposal"
}

Returns: { "score": 78, "grade": "B", "recommended_actions": [...] }

SEO Factory — Generate

POST /api/products/seo-factory/generate
{
