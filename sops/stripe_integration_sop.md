# SOP — Stripe Integration Service

## Service: Stripe Payment Integration

### Project Brief
Integrate Stripe payments, webhooks, and billing into client application.
Estimated timeline: 2-3 business days.
Price tier: $1,500 flat.

### Deliverables Checklist
- [ ] Stripe account connected (client's or new)
- [ ] Products and prices created in Stripe dashboard
- [ ] Webhook endpoints deployed and verified
- [ ] Test mode payments working end-to-end
- [ ] Live mode payments working end-to-end
- [ ] Revenue allocation rules configured (if included)
- [ ] Accounting sync configured (if included)
- [ ] Client dashboard access confirmed

### Handoff Checklist
- [ ] Stripe dashboard access confirmed for client
- [ ] Webhook signing secrets rotated and secured
- [ ] Test vs live key separation confirmed
- [ ] Invoice sent
